Υποστήριξη Harmonic Hands

Βασική λειτουργία ανά σελίδα (Οριζόντια καρτέλα)

  1. Έναρξη και διακοπή αναπαραγωγής

    Πατήστε τον λαγό για να ξεκινήσετε την αναπαραγωγή και το κυματοειδές σχήμα για να τη διακόψετε. Χρησιμοποιήστε την κορώνα για να ρυθμίσετε την ένταση.

  2. Ρύθμιση παραμέτρων συνθεσάιζερ

    Ελέγξτε τις παραμέτρους δύο ξεχωριστών ταλαντωτών για να τους συνδυάσετε και να δημιουργήσετε τον μοναδικό σας ήχο.

  3. Ρυθμίσεις συντονισμού

    Ενεργοποιήστε ή απενεργοποιήστε το autotune και/ή ρυθμίστε το εύρος συχνοτήτων για να ελέγξετε τις νότες που θα παράγει ο συνθεσάιζερ.

  4. Παράμετροι κίνησης

    Ρυθμίστε τις παραμέτρους κίνησης ώστε να προσαρμόζονται στο στυλ παιχνιδιού σας.

Συμβουλές και κόλπα

  1. Αποτροπή ύπνου εφαρμογής κατά τη περιστροφή του καρπού:

    Ρυθμίστε την επιλογή "Επιστροφή στο Ρολόι" στο WatchOS:

    Αλλάξτε την επιλογή "Επιστροφή στο Ρολόι" σε "2 λεπτά" ή "1 ώρα", διαφορετικά η εφαρμογή θα μεταβεί σε κατάσταση ύπνου όταν ο καρπός σας φτάσει σε μια συγκεκριμένη γωνία.

  2. Ρυθμίστε την ένταση χαμηλά πριν την έναρξη της εφαρμογής

    Αποφύγετε την ανεπιθύμητη ένταση και προστατέψτε την ακοή σας, περιστρέφοντας την κορώνα προς τα κάτω στην πρώτη οθόνη ώστε η ένταση να ξεκινά χαμηλά.

Γνωστά προβλήματα

  1. Ο ήχος μπορεί να διακοπεί

    Αυτό δεν θα πρέπει να συμβαίνει συχνά, αλλά αν συμβεί, η εφαρμογή θα επανεκκινήσει αυτόματα τη μηχανή ήχου. Εξετάζουμε το θέμα για να το βελτιώσουμε.

  2. Βελτιστοποίηση διάρκειας μπαταρίας

    Δεδομένου ότι η εφαρμογή χρησιμοποιεί ορισμένους πόρους όταν είναι ενεργή, η μπαταρία θα εξαντλείται πιο γρήγορα από το να είναι σε αδρανή κατάσταση. Εξετάζουμε αν έχει νόημα να προσθέσουμε μια λειτουργία βελτιστοποίησης μπαταρίας, επιτρέποντας την απενεργοποίηση ορισμένων λειτουργιών.
