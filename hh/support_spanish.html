Soporte para Harmonic Hands

Operación básica por página (Pestaña horizontal)

  1. Iniciar y detener la reproducción

    Toca el conejo para iniciar la reproducción y la onda para detenerla. Usa la corona para ajustar el volumen.

  2. Ajustar los parámetros del sintetizador

    Controla los parámetros de dos osciladores independientes para combinarlos y generar un sonido único.

  3. Configuración de afinación

    Activa o desactiva el autotune y/o ajusta el rango de frecuencias para controlar las notas que generará el sintetizador.

  4. Parámetros de movimiento

    Ajusta los parámetros relacionados con el movimiento para adaptarlos a tu estilo de interpretación.

Consejos y trucos

  1. Evitar que la aplicación se suspenda al girar la muñeca:

    Ajusta la configuración de "Volver al reloj" en WatchOS:

    Cambia la configuración de "Volver al reloj" a "2 minutos" o "1 hora", de lo contrario, la aplicación se suspenderá cuando tu muñeca alcance cierta posición.

  2. Baja el volumen antes de iniciar la aplicación

    Evita un volumen no deseado y protege tu audición girando la corona hacia abajo en la primera vista para comenzar con un volumen bajo.

Problemas conocidos

  1. El sonido puede interrumpirse

    Esto no debería ocurrir con frecuencia, pero si sucede, la aplicación reiniciará automáticamente el motor de sonido. Estamos investigando este problema para mejorarlo.

  2. Optimización de la duración de la batería

    Debido a que esta aplicación utiliza algunos recursos mientras está activa, la batería se descargará más rápido que en estado de reposo. Estamos evaluando si es viable agregar un modo optimizado para batería, permitiendo desactivar ciertas funciones.
